Making Instance Methods in Ruby Class Definitions

The Lowdown

If you have looked into Ruby at all you probably are aware of class methods, or things that do things to different objects. For example arrays and hashes have their own methods that you can call on any array or hash and the object will understand what to do. For example you can call array.map or array.each. But what if you want to make your own class and have it do something? That's where instance methods come in.

Creating an Instance Method

  • Start by using def and the name of your desired method.
  • If you need any user input, pass it in in ().
  • Fill in the method with the functions of your choice. Maybe you want some loops to perform an action on your input, or maybe you simply want it to return something.
  • Be sure to close it out with end.
  • Once you've created your method within the Class definition, you can use it on any new class object that you create!
